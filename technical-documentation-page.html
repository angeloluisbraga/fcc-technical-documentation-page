Introduction

HTML stands for HyperText Markup Language.

It is the standard language for creating web pages.

<!DOCTYPE html>
  • Easy to learn
  • Works in every browser
HTML Elements

Elements define the structure of a webpage.

Most elements have opening and closing tags.

<h1>Hello</h1>
  • Heading
  • Paragraph
Attributes

Attributes provide additional information.

They are placed inside the opening tag.

<img src="image.jpg" alt="Example">
  • src
  • href
Lists

HTML supports ordered and unordered lists.

Lists help organize content.

<ul><li>Item</li></ul>
  • Ordered list
  • Unordered list
Forms

Forms collect user input.

Input fields can have many different types.

<input type="text">
  • Text input
  • Submit button
